📋 文件复制助手 / File Copy Helper

点击"复制"按钮，然后到GitHub对应文件粘贴即可 / Click copy, then paste into GitHub file editor

操作步骤 / Steps

1. 打开 GitHub仓库

2. 进入对应文件，点 ✏️ 铅笔图标进入编辑模式

3. Ctrl+A 全选旧内容 → Delete 删除

4. 点击下方按钮复制新内容 → Ctrl+V 粘贴

5. 点 Commit changes 保存
