# 飞书多维表格配置指南

## 📋 快速开始

### 第一步：创建飞书多维表格

1. 打开飞书，进入"多维表格"应用
2. 点击"新建多维表格"
3. 命名为：`🦐 海青水产产品分析数据库`
4. 记录下表格的 `app_token`（URL中的最后一部分）

### 第二步：创建数据表

在多维表格中创建以下5张数据表：

#### 表1：产品基础信息

#### 表3：产品研发

#### 表4：菜品应用

### 第三步：配置环境变量

**方法1：在代码中设置**
```python
import os
os.environ["FEISHU_APP_TOKEN"] = "your_app_token_here"
```

**方法2：在系统环境变量中设置**
```bash
# Linux/Mac
export FEISHU_APP_TOKEN="your_app_token_here"

# Windows
set FEISHU_APP_TOKEN=your_app_token_here
```


## ⚠️ 常见问题

### Q1: 提示"未配置飞书多维表格app_token"
**解决方案**：
- 检查环境变量 `FEISHU_APP_TOKEN` 是否正确设置
- 确保 `app_token` 没有多余的空格或引号

### Q2: 提示"飞书多维表格录入失败"
**解决方案**：
- 检查表格名称是否匹配（必须完全一致）
- 检查字段名称是否匹配（必须完全一致）
- 确认你有该多维表格的编辑权限

### Q3: 数据没有录入
**解决方案**：
- 检查工作流是否完整执行
- 查看日志中的错误信息
- 确认表格结构符合要求

## 💡 使用建议

1. **产品ID规则**：系统自动生成格式为 `PRD{时间戳}` 的唯一ID
2. **字段长度限制**：飞书文本字段最大支持10000字符，超出会被截断
3. **日期格式**：自动使用毫秒时间戳格式
4. **多表关联**：通过产品ID关联不同表的数据

## 📊 数据查询示例

在飞书多维表格中，你可以：

2. **按产品类型分类**：查看鳕鱼、三文鱼等不同品类的分析

4. **创建视图**：为不同团队创建定制化视图

相比文本消息推送，多维表格方案具有以下优势：

✅ **可编辑**：随时修改和补充分析结果  
✅ **可导出**：支持导出为Excel等格式  
✅ **协作友好**：团队成员可同时查看和编辑  
✅ **历史追溯**：保留所有历史记录  
✅ **数据统计**：支持汇总和统计功能
